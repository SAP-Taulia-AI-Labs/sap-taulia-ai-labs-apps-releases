# SAP Taulia AI Labs - App Releases

Public release manifests and binaries for SAP Taulia AI Labs applications.

## Structure

Each application has its own directory containing:
- `manifest.json` — version info and download URL (fetched by apps for update checks)
- Release zip files

## Applications

- **claude-menu-bar/** — Claude Menu Bar Lite for macOS
